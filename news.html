Release v1.0.1.1.0 - First Stable Version

We are excited to announce the release of Lime3D version v1.0.1.1.0! This release includes performance improvements, bug fixes, and the first implementation of the core GPU rendering engine. It's now stable and ready for use in your projects.

Release Date: February 20, 2025

New Features in Lime3D v1.0.1.1.0

Lime3D v1.0.1.1.0 brings multiple enhancements to improve rendering performance and API compatibility. Several new rendering techniques have been introduced, including support for basic lighting and shadow rendering. We have also made the API more stable and consistent.

Release Date: February 20, 2025

Upcoming: OpenGL API Support

Lime3D is actively working on adding support for the OpenGL API. This update will allow you to integrate Lime3D into your OpenGL-based applications, enabling more graphics-related features and compatibility across different platforms. The update is expected to be available in the next version of Lime3D.

Expected Release Date: March 15, 2025
