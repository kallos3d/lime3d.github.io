Getting Started

Welcome to Lime3D! Lime3D is an open-source GPU rendering library that allows you to integrate GPU acceleration in your applications. Below is a basic introduction to set up and use Lime3D.

Prerequisites

Make sure your development environment is ready with the following:

  • Linux or Windows operating systems supported.
  • GNU Compiler Collection (GCC) or Clang for Linux/Mac, Visual Studio for Windows.
  • CMake for building the project.
  • Make for building the project.

Step 1: Clone the Repository

First, clone the Lime3D repository using Git: 

git clone https://github.com/kallos3d/lime3d.git

This will download the entire Lime3D project to your local machine.

Step 2: Install Dependencies

You need to install a few dependencies to ensure Lime3D works correctly. These are necessary for building and running the project. If you're on Linux, you can install them using apt:

sudo apt-get install build-essential cmake libglew-dev

If you're on Windows, install **MinGW** or **Visual Studio** for compiling, and ensure **CMake** is installed.

Step 3: Build Lime3D

Once you've cloned the repository and installed dependencies, you can now build Lime3D:

cd lime3d
mkdir build && cd build
cmake ..
make

The `make` command will compile Lime3D using GCC, ensuring all components are built.

Step 4: Install Lime3D

After building, you can install Lime3D on your system with:

sudo make install

Installation

Getting Lime3D Installed on Your System

Lime3D can be easily installed on your system after cloning the repository and setting up your development environment.

Installation Steps

  • Ensure all dependencies are installed. If using Linux, use apt for installing packages like build-essential, CMake, and GLEW.
  • Clone the repository using Git:
    • git clone https://github.com/kallos3d/lime3d.git
  • Navigate to the directory and create a build directory:
    • cd lime3d
mkdir build
  • Run CMake to generate necessary build files:
    • cmake ..
  • Now, compile the source code with:
    • make
  • To install Lime3D system-wide, use:
    • sudo make install

Getting Lime3D into Code

Once you have Lime3D installed and compiled, you can start integrating it into your code. This section will show you how to link Lime3D to your project.

Step 1: Include Lime3D in Your Project

To use Lime3D in your code, include the necessary header files in your source code:

#include

Step 2: Initialize Lime3D

You need to initialize Lime3D before you can use its functions. Add this line of code at the beginning of your program:

Lime3D::init();

Step 3: Start Using Lime3D Functions

Now you're ready to call Lime3D functions. For example, to render a 3D object:

Lime3D::render();
